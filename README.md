\# HealthPlat

Android health monitoring dashboard (MVP) built with Jetpack Compose \& Clean Architecture.



\### Modules

\- app: UI and presentation layer

\- data: repositories \& API

\- domain: use cases \& models

\- blesdk: BLE SDK wrapper for Bonlala device



\### Tech Stack

Kotlin, Jetpack Compose, MVVM, Hilt, Retrofit, Coroutines, Bonlala BLE SDK, Nordic DFU
